This project is a fully functional E-Commerce platform designed with separate environments for users and administrators. The user interface is built using React.js, providing a smooth, dynamic, and responsive experience. The platform allows customers to browse products, view details, and interact with a modern UI.

The Admin Panel includes complete CRUD operations for managing products and users, implemented using Local Storage for data handling in the frontend environment. Administrators can add, edit, update, or delete product data through a clean and user-friendly dashboard.

Tech Stack Used: React.js, Bootstrap
Key Features: Dynamic UI, Responsive Design, Product Management, User Management, CRUD Functionality
